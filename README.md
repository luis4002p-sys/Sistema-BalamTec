# Sistema-BalamTec

Aplicación interna para la gestión de ventas, clientes, empleados y productos.  
Construida con tecnologías web pero usada como sistema de escritorio interno.

## Tecnologías utilizadas
- HTML, CSS, JavaScript
- Node.js (backend)
- Sistema tipo escritorio ejecutado en navegador interno

## Descripción
Sistema pensado para uso interno en la empresa, sin funciones de comercio electrónico, solo administración.
